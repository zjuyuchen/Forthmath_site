常用命令

x <- c(1,3,2,5) 或者 x = c(1,3,2,5)将直接给定x 1 3 2 5这个向量。
length(x) 返回x的长度
ls() 显示当前所有对象
rm（x） 删除x对象
rm （list = ls()） 删除所有对象
sqrt(x) 对所有数据求开根
x^2 对所有数据求平方
x = rnom() 产生一个正态分布的随机向量
y=x+rnorm (50, mean=50, sd=.1) 50个数字，平均值50，标准差0.1
set.seed(1001) 1001是种子的编号，下次调用set.seed(1001)会产生同样的分布
plot(x,y) 画图函数
plot(x,y,xlab="x-axis",ylab="y-axis",main=" Plot of X vs Y") 画图详细参数
pdf('Figure1.pdf') plot(x,y) dev.off() 三个命令在一起可以画图并导出在pdf里面。
seq(1,10，length=10) 产生10个1~10之间的等差数列
f = f=outer(x,y,function (x,y)cos(y)/(1+x^2)) 自定义函数
contour (x,y,f) 画contour图。 f为x,y的函数
image(x,y,fa) 根据fa数值画数据
persp(x,y,fa ,theta =30, phi =40) 三维图，theta, phi控制看的角度
A = matrix (data=c(1,2,3,4) , nrow=2, ncol =2) 建立一个2*2的矩阵
B = matrix (c(1,2,3,4) ,2,2,byrow =TRUE) 先按照row来排列数据
A[2,3] 选取A中2行3列的数据
注意，R语言中数组标记以1开始
A[c(1,3) ,c(2,4) ] 选取1,3行，2,4列的数据
A[1:3 ,2:4] 1~3行，2~4列的数据
A[-c(1,3) ,] 选取除了1,3行的所有数据
dim(A) 返回A的维度

数据I/O
Auto=read.table ("Auto.data ") 读取table数据
fix(Auto) 打开窗口看该数据
Auto=read.table ("Auto.data", header =T,na.strings ="?") 告诉read数据有表头，而且‘？’的地方是na
Auto=read.csv (" Auto.csv", header =T,na.strings ="?") 读取CVS文件
dim(Auto) 显示数据维度
names(Auto) 显示数据的列名，variable names
plot(Auto$cylinders , Auto$mpg ) 画出Auto数据中的两个列
attach (Auto) plot(cylinders , mpg) 分两步画出来也可以
cylinders =as.factor (cylinders ) 当数据分布很小时，可以转化为定性数据
plot(cylinders , mpg , col ="red", varwidth =T, xlab=" cylinders ",ylab ="MPG ")
hist(mpg ,col =2, breaks =15) 柱状图，col=2 就是红色
pairs(Auto) 画出Auto中两两列的所有组合图
identify (horsepower ,mpg ,name) 只显示满足name的点
summary (Auto) 对数据做个统计总结。
